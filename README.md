# Personal Website

Static personal website for teaching, research, and entrepreneurial activities.

## Publish With GitHub Pages

1. Create a new GitHub repository named `personal-website`.
2. Upload these files to the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `.nojekyll`
   - `README.md`
3. Open the repository on GitHub.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`, set:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your site will be available at:

`https://YOUR-GITHUB-USERNAME.github.io/personal-website/`
